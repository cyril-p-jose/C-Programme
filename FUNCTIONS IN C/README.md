# 📘 C Programming Practice folder

A collection of **basic C programming programs** created for learning and practicing core programming concepts.

---

## 📂 Program Categories

The programs in this folder are organized into the following categories:

### 1️⃣ Common Programs

Basic programs used to understand fundamental programming logic.

Examples include:

* Number-based programs
* Array operations
* Matrix operations
* Basic string handling
* Sorting algorithms

---

### 2️⃣ Programs Using Functions

Programs that demonstrate how to divide large programs into **smaller reusable functions**.

Concepts covered:

* Function declaration
* Function definition
* Function call
* Functions with arguments
* Functions with return values
* Call by value
* Call by reference

---

### 3️⃣ Programs Using Recursion

Programs that use **recursive functions**, where a function calls itself until a base condition is reached.

Concepts covered:

* Recursive function calls
* Base condition
* Problem solving using recursion

---

### 4️⃣ Programs Using Structures

Programs that demonstrate the use of **structures to store related data together**.

Concepts covered:

* Structure declaration
* Structure variables
* Array of structures
* Accessing structure members

---

## 🧠 Concepts Practiced

This folder helps in practicing important **C programming fundamentals**, including:

* Variables and Data Types
* Conditional Statements
* Loops
* Arrays
* Strings
* Functions
* Recursion
* Structures

---
